## 📝 自定义指南

### 修改品牌色
编辑 `src/App.vue` 中的 CSS 变量：
```css
:root {
  --color-brand: #8B5E3C;  /* 修改这里 */
  --color-brand-dark: #6B4226;
  --color-brand-light: #C4A882;
}
```

### 修改字体
编辑 `index.html` 中的 Google Fonts 链接：
```html
<link href="https://fonts.googleapis.com/css2?family=YOUR_FONT&display=swap" rel="stylesheet" />
```

### 添加新页面
1. 在 `src/pages/` 中创建 `NewPage.vue`
2. 在 `src/router/index.js` 中添加路由：
```javascript
{
  path: '/new-page',
  component: () => import('../pages/NewPage.vue'),
}
```
3. 在导航中添加链接

### 配置 EmailJS
编辑 `src/pages/Contact.vue`：
```javascript
emailjs.init('YOUR_PUBLIC_KEY')
await emailjs.send('SERVICE_ID', 'TEMPLATE_ID', data)
```
